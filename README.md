# Tic Tac Toe Tomatetoe Tomauto

### Five Tea Chicken from Tennessee

Raphi Ramot and Vanna Lei

### Project Description:

This project is a Tic Tac Toe Game using what we learned in Stuyvesant's System Level Programming class. Coded using C, you can connect to other servers, be matched with a random player, or play against your friend on the same computer. Enjoy!!

### Instructions:

How does the user install/compile/run the program. (CHANGE THIS!!!!!)
Type XX in terminal to start.

How does the user interact with this program? (CHANGE THIS!!!!!)
1. Once connected to an opponent and prompted to start the game, type y and n to confirm.
2. To play the game, type the coordinates of an empty box.
    Top (T), Middle(M), Bottom (B), Left (L), Right(R)
    TL | TM | TR
    ———|————|———
    ML | MM | MR
    ———|————|———
    BL | BM | BR
3. If already taken, you will be prompted to choose a different spot.
4. Wait for your opponent's turn and repeat.
5. Have your pieces three in a row or diagonally to win.
6. When the game is over, you will have the option to play again with the same client, never play again with the same client, or connect with a random person.
